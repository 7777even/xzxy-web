## 快速开始

1. 安装依赖
   
npm install

2. 运行开发环境

npm run dev

3. 构建生产版本

npm run build


## 开发规范

1. 代码风格
- 遵循 ESLint 规则

2. 组件开发
- 采用函数式组件,保持组件函数特性
- 使用 React Hooks

3. 样式管理
- 使用 Tailwind CSS
- 遵循响应式设计
